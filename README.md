# 📡 BLE Logger

**Bluetooth logger and debug playground for micro:bit** — connect, inspect services, monitor characteristics and log data in real time.

## Quick Start

Open `index.html` in a Web Bluetooth-compatible browser (Chrome/Edge).

## Files

| File | Description |
|------|-------------|
| `index.html` | Main app |
| `script.js` | BLE connection and logging logic |
| `style.css` | UI styles |
| `makecode.ts` | MakeCode companion code for micro:bit |
| `logo.svg` | App logo |

## Features

- Connect to micro:bit via Web Bluetooth
- Inspect BLE services and characteristics
- Real-time data logging and hex display
- Debug playground for BLE development

## Tech

HTML · CSS · JavaScript · Web Bluetooth API · MakeCode
